# Sekolah Tanah Air

Proyek ini adalah website untuk platform *crowdfunding* (donasi) khusus inisiatif "Sekolah Tanah Air".

## Menjalankan secara Lokal

**Prasyarat:** Node.js

1. Instal dependensi:
   ```bash
   npm install
   ```
2. Salin `.env.example` ke `.env.local` (sesuaikan konfigurasi Supabase).
3. Jalankan aplikasi:
   ```bash
   npm run dev
   ```
